# tkinter_kouza

# 講座一回目
+ gui.py

「ボタンを押すウィンドウ」を生成します。

+ gui_after.py

ボタンを押すと、入力フォームに入れた文字を表示させます。

+ gui_enshu.py

演習問題の回答です。（ボタンが3つ配置されています。）

# 講座二回目
+ save.py

「ボタンを押すウィンドウ」を生成します。

+ save_after.py

ボタンを押すと、保存と読み込みができるウィンドウを生成します。

+ save_enshu.py

演習問題の回答です。（入力フォームが3つあるウィンドウを生成します。）

# 講座三回目
+ graph.py

「ボタンを押すとグラフを描画する」ウィンドウを生成します。

+ led.py

「ボタンを押すとLEDを点灯する」ウィンドウを生成します。
